# test-junie

Este es un repositorio de prueba para interactuar con Junie, el asistente de IA desarrollado por JetBrains.

## Descripción
Este repositorio se utiliza para validar y demostrar las capacidades de Junie en la automatización de tareas de desarrollo, gestión de issues y flujos de trabajo de GitHub.
